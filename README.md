# novacrust_accessment

Frontend Developer Take-Home Assessment (React / Next.js)

## Project summary (in one sentence)

A clean, component-driven implementation of a crypto exchange experience (check rate -> convert -> ...), built with Next.js and TypeScript, optimized for speed, maintainability, and a pleasant developer experience.

## Local setup

### Prerequisites: Node 18+ and your package manager of choice (npm, pnpm, or yarn)

#### install dependencies

pnpm install

#### run development server

pnpm dev

#### build for production

pnpm build

#### run the production build locally

pnpm start

If you prefer npm or yarn, replace pnpm with npm or yarn in the commands above.
